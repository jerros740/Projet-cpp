Pour lancer le programme : 
- Se placer dans le répertoire ProjectSDL_Part1_base
- Dans le powershell, et après avoir compiler, taper(chemin de l'executable) : out\build\x64-Debug\Project_SDL_Part1_base\SDL_part1.exe <n_sheep n> <n_wolf> <duration_seconds>
- Pour jouer, utiliser le touche directionnelle du clavier (haut,bas,gauche,droite)
